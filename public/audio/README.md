# Audio Files for SalahSteps Component

This directory contains audio files for the interactive prayer guide.

## Required Audio Files:

1. `niyyah.mp3` - Intention recitation
2. `allahu-akbar.mp3` - Takbeer (Allahu Akbar)
3. `fatihah.mp3` - Surah Al-Fatihah recitation
4. `ruku.mp3` - Ruku supplication
5. `standing-from-ruku.mp3` - Standing from ruku
6. `sujud.mp3` - Sujud supplication
7. `tashahhud.mp3` - Tashahhud recitation
8. `salam.mp3` - Final salam

## Audio Guidelines:

- Format: MP3
- Quality: High quality (192 kbps or higher)
- Language: Arabic with clear recitation
- Duration: Appropriate for each prayer step
- Volume: Consistent levels across all files

## Recording Notes:

- Use a clear, professional recitation
- Ensure proper Arabic pronunciation
- Include appropriate pacing for learning
- Consider adding brief pauses between phrases

## Usage:

These audio files are automatically loaded by the SalahSteps component and played when users click the play button for each step.
